Задание:
Создать телефонный справочник с возможностью импорта и экспорта данных в нескольких форматах.
1)"model" - модули исполнения (пользовательские функции)
2)"view" - интерфейс взаимодействия с пользователем
3)"controller" - соединяет в себе функционал "view"  и  "model"

Enter the number corresponding to the action:
    1 - Show phone book on screen
    2 - Add new entry
    3 - Write data to csv format
    4 - Write data to json format
